# kubsh - Custom Shell

Шелл с расширенными возможностями для Hack the Box.

## Функции

1. Базовый ввод/вывод
2. История команд с сохранением в ~/.kubsh_history
3. Встроенные команды:
   - `echo <text>` - вывод текста
   - `\e $VAR` - вывод переменной окружения
   - `\l /dev/sda` - информация о разделах диска
   - `\adduser username` - добавление пользователя
   - `\deluser username` - удаление пользователя
   - `\q` - выход из шелла
4. Поддержка внешних команд и пайпов
5. Обработка сигнала SIGHUP
6. VFS с информацией о пользователях в ~/users/

## Сборка

```bash
make compile      # Компиляция
make run          # Запуск
make install      # Установка в /usr/local/bin
make deb          # Создание deb-пакета
make clean        # Очистка
